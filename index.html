File Metadata Microservice

by Kevin Madill (http://www.github.com/kmadill)


This simple API was created as part of Free Code Camp's curriculum. The exercise can be found here. If you wish to view the source code, you can find that here.

User stories:

  1. I can submit a FormData object that includes a file upload.
  2. When I submit something, I will receive the file size in bytes within the JSON response.

Give it a try!

Use the form below to select a file. When you submit, you will be returned JSON with the size of the file you submitted.
